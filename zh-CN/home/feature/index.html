Features

Data Catalog


Automatically obtain datasource metadata and construct enterprise-level data catalog; Regularly monitor data schema changes.

Data Quality Monitor


Multiple data quality check rules are built in the platform, which can be used out of the box; Provide alarm message management to help you better grasp the quality situation.

Microkernel & Plug-in


With plug-in design as the core, the platform supports plug-in design of datasources, rules, alarm components, error data storage and registry.

Data Profile


Data profile enables you to understand the situation of each data set faster and more easily, so that you have a clear idea.

Robust and stable


Created by big data experts. Well-designed. Complete test cases.

Multiple Mode


DataVines not only provides a platform to help you observe data, but also supports local mode integration into the workflow of the scheduling system.
